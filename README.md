# 📦 Складская утилита

Полнофункциональная веб-система для автоматизации учета складских запасов, генерации и сканирования штрихкодов, управления размещением товаров по ячейкам/стеллажам, а также проведения инвентаризации в реальном времени.

---

## 🚀 Основные возможности

### 1. 📋 Управление товарами и остатками
* **База товаров:** Название, артикул, категория, бренд, количество, минимальный остаток, цена, статус, ячейка/стеллаж хранения.
* **Импорт и экспорт данных:**
  * Загрузка и обновление данных из файлов `.xlsx`, `.xls`, `.csv`.
  * Выгрузка актуального справочника и остатков в файл Excel (`.xlsx`).
* **Поиск и фильтрация:** Быстрый поиск по наименованию, артикулу, штрихкоду, а также фильтрация по категориям, статусам остатков и локациям.

### 2. 🏷️ Генерация и обработка штрихкодов
* **Генерация штрихкодов:** Автоматическое создание штрихкодов формата EAN-13, Code128 для товаров и складских ячеек.
* **Печать этикеток:** Подготовка к печати термоэтикеток с наименованием, артикулом, ячейкой и штрихкодом.
* **Сканирование штрихкодов:**
  * Поддержка аппаратных сканеров (USB / Bluetooth ТСД в режиме эмуляции клавиатуры).
  * Интегрированное сканирование через камеру устройства (смартфона, планшета или ноутбука).
  * Быстрый поиск товара или ячейки при сканировании.

### 3. 📍 Складская логистика и адресное хранение
* **Адресное хранение:** Привязка товаров к конкретным стеллажам, полкам и ячейкам (например, `A-01-02`).
* **Приход / Расход / Перемещение:** Быстрое изменение количества товаров с указанием типа операции и комментариев.
* **Журнал операций (Activity Log):** Полная история изменений, перемещений, приходов и списаний с фиксацией времени и автора.

### 4. 🎨 Интерфейс и кастомизация (Темная тема)
* **Переключение тем:** Поддержка светлой и темной тем оформления интерфейса с мгновенным переключением кнопкой в шапке панели и сохранением выбора пользователя.
* **Адаптивные элементы управления:** Оптимизированные стили для всех полей ввода, выпадающих списков, кнопок и модальных окон для комфортной работы при любом освещении.

### 5. 📊 Журнал изменений и безопасность
* Сохранение логов операций до 14 дней с возможностью фильтрации по датам и типам событий.
* Автоматическая валидация вводимых данных и защиты от дублирования артикулов и штрихкодов.
* Функционал отмены последнего действия (**↺ Отменить**) прямо из панели управления.

---

## 🛠️ Технологический стек

* **Backend:** Node.js, Express.js
* **Database:** SQLite / JSON-база (локальное хранилище данных `db.js`)
* **Frontend:** HTML5, CSS3 (Modern Vanilla UI / CSS Variables / Light & Dark Themes), JavaScript (ES6+)
* **Libraries & Tools:**
  * `JsBarcode` — генерация штрихкодов на стороне клиента.
  * `Html5-QRCode` / `ZXing` — сканирование штрихкодов с камеры.
  * `XLSX (SheetJS)` — обработка Excel/CSV файлов.
